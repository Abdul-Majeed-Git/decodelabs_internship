======================================
       General Knowledge Quiz Game
======================================

Developer | Abdul-Majeed
Language  | Python 3.12

============
Description:
============
This is a simple and fun Quiz Game made with Python. 
The program asks the user 3 questions, gives points for 
correct answers, and shows the total score at the end.

=========
Features:
=========
->| Smart Answer Check: If the user types in CAPITAL letters or adds extra spaces, 
    the program automatically cleans the text and checks the answer correctly.
->| Uses Functions: The code uses a function (def) to keep everything clean, neat, and organized.
->| Score Counter: Every time you give a correct answer, the score automatically increases by 1 point.
->| Clean Layout: The final score is displayed inside a beautiful box at the end of the game.

=======================
How to Run the Project?
=======================
1. Make sure Python is installed on your computer.
2. Open this project folder in VS Code.
3. Open the Terminal and type this command, then press Enter:
   python quiz_game.py

============
How to Play?
============
1. When the game starts, you will see a Welcome message on the screen.
2. Questions will appear one by one. Type your answer right after the 'Ans:' prompt.
3. Press Enter after typing. The system will immediately tell you if your answer is 'Correct' or 'Wrong'.
4. After all 3 questions, you will see the 'GAME OVER' screen with your Final Score.
